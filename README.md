🔗 **Live Website:** [https://inferr.dev](https://inferr.dev)

---

### 🚀 Local Deployment
If you want to view or test this portfolio on your own machine:

1. Clone this repository to your local computer.
2. Open your terminal inside the project directory.
3. Start a local web server (needed for the contact form functionality to work):
   ```bash
   python -m http.server 8000
   ```
4. Open your browser and navigate to `http://localhost:8000`.
